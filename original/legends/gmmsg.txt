I recommend Programmer's File Editor for creating scripts. There are

        2. Type /usr/home/azure/pfe

        3. If you want the 32 bit version, type sz pfe0602i.zip
           If you want the 16 bit version, type sz pfe0602.zip

There is a directory called scripts in your home directory on sutr.
This is where all of your script files should be kept. Only you and
users in group NORMAC can enter this directory. Script files MUST
end in .SCR in order for them to be compiled by the Cafe.

To compile a script in the Cafe, you use the @OLDCOMP command. When
specifying the script file to compile, you leave off the .SCR. So if
you want to compile a script called myoffice.scr, you would type...

        @old scripts/myoffice
